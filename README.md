# Cinematic Video Editor Portfolio üé¨

Hi! This is the source code for my personal professional video editing portfolio.

My main goal with this project was to convey **cinematic quality** from the very first second. I didn't want a boring, static website. I wanted something that felt alive, premium, and visually striking‚Äîjust like my edits.

## What makes it special?

Everything you see here was hand-coded by me. I initially considered using complex frameworks, but I decided to go back to basics to have full control:

- **100% Vanilla**: Just HTML, CSS, and JavaScript. No weird installations, no loading times. Double-click and it works.
- **"Cinema" Design**: Deep dark backgrounds, gradients that look like spotlights, and glassmorphism effects in the menus.
- **True Responsiveness**: It looks insane on PC with the full-screen carousel, but on mobile, I adapted everything so it can be navigated perfectly with one hand.

## How I built it

It was a step-by-step process (and a lot of fun):

1.  **The Cinematic Base**: First, I defined the color palette (deep blacks, electric blue, neon purple) and modern typography.
2.  **The Infinite Carousel**: Probably the hardest part. I wanted the images to **fade** smoothly into each other rather than slide, and to occupy the entire background.
3.  **Dropping Frameworks**: I started experimenting with Vite and Tailwind, but I realized that for a portfolio like this, they were adding more noise than value. So... out they went! I migrated everything to clean code, and now it flies.
4.  **Details Matter**:
    - The menu button that looks like an audio equalizer dancing.
    - The text with soft gradients to ensure readability over any image.
    - The project cards with that subtle glow when you hover over them.

## How to use it?

If you like the code, it's all yours:

1.  Download it.
2.  Open the `index.html` file in your browser.
3.  That's it!

If you want to edit it, you'll see that I organized the CSS by sections (Header, Hero, Colors...) so you can find everything quickly.
